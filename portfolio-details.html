Chatten - Chat App with OTP Authentication

Chatten is a modern Android chat application that supports phone number OTP-based authentication, user profile setup, and a visually polished UI inspired by leading messaging apps. It includes chat, call, and status features organized in a clean tabbed interface. Built in Java with Firebase integration for backend services, it emphasizes UI/UX quality.
