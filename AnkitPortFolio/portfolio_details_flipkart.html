Flipkart Review Sentiment Analysis

A sentiment analysis project on Flipkart product reviews, built using Python with NLTK and Pandas. It includes text preprocessing, polarity classification (positive, negative, neutral), and visualization using Seaborn and Matplotlib. The goal was to extract actionable insights from user-generated feedback.
